### 通信コマンド仕様

| No. | Command Name   | Details                                |
|-----|----------------|----------------------------------------|
|   1 | CAT_MOVE       | キャタピラーのモータを制御する。       |
|   2 | CAT_SLOW_MODE  | キャタピラーのスローモードを制御する。 |
|   3 | HEATER_MODE    | ヒーターの通電を制御する。             |
|   4 | ARM_MOVE       | アームのモータを制御する。             |
|   5 | EMERGENCY_STOP | 緊急停止用。                           |

*Packet Format*

```
 0             15               31
+---------------+----------------+
|          Command No.           |
+---------------+----------------+
|            Option              |
+---------------+----------------+
```

* Command No. : コマンド番号
* Option : コマンドのオプションを指定
